# 📊 Customer Churn Prediction App

## 🚀 Project Overview

This project predicts whether a customer is likely to **churn (leave the service)** or not using Machine Learning.

The model is deployed as a **web application using Streamlit**, where users can input customer details and get instant predictions.

---

## 🎯 Objectives

* Analyze customer data
* Build a predictive model for churn
* Deploy the model as an interactive web app

---

## 🧠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Imbalanced-learn (SMOTE)
* Streamlit

---

## ⚙️ Machine Learning Workflow

1. Data Cleaning & Preprocessing
2. Handling Missing Values
3. Encoding Categorical Features
4. Balancing Data using SMOTE
5. Feature Scaling using StandardScaler
6. Model Training using Random Forest Classifier

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/customer-churn-app.git
```

2. Navigate to the folder:

```
cd customer-churn-app
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
